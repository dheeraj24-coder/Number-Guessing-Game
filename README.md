# Number-Guessing-Game
🎯 Number Guessing Game (Python)

This is a simple Python-based number guessing game where the computer randomly selects a number between 1 and 20, and the user has to guess it correctly.

🚀 Features
Random number generation using Python’s built-in random module
User-friendly hints like “thoda bada soch” or “thoda chota soch”
Counts the number of attempts taken to guess correctly
🛠️ How It Works
The computer generates a random number between 1 and 20
The user inputs a guess
If the guess is:
Less → Hint to guess higher
More → Hint to guess lower
The loop continues until the correct number is guessed
▶️ How to Run
Install Python
Copy the code into a .py file
